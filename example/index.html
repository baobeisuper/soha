SOHA

SOHA is an enhanced library of Go(Golang) templates，it extends the capabilities of the Go template engine by custom function。

SOHA是一个Go语言(Golang)模板的增强库，它通过自定义函数的方式扩展了Golang自带的模板引擎能力，让我们可以更好的开发Go Web应用。

Prerequisite

  1. Go version 1.13+
  2. Using Go Modules

SOHA Example

int

Example:{{`{{int "1" |printf "%T"}}`}}
Output:{{int "1" |printf "%T"}}

string

Example:{{`{{string 1 |printf "%T"}}`}}
